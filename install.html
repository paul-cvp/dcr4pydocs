Installation#

pip#

To use pm4py on any OS, install it using pip:

(.venv) $ pip install pm4py

pmp4y uses the Graphviz library for rendering visualizations. Please install Graphviz.

After installation, GraphViz is located in the program files directory. The bin\ folder of the GraphViz directory needs to be added manually to the system path. In order to do so, please follow this instruction.

Docker#

To install pm4py via Docker, use:

$ docker pull pm4py/pm4py-core:latest

To run pm4py via docker, use:

$ docker run -it pm4py/pm4py-core:latest bash
